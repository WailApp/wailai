Frequently Asked Questions

Why doesn't WailAI work in my browser?

The reason WailAI is not working might be because your browser is outdated. If your version is listed below:

Internet Explorer

Internet Explorer 11

Opera 12

Old Microsoft Edge

Old Google Chrome (below 50)

Old Safari (below 10)

Old Firefox (below 50)

You should update your browser to the latest version. If your device doesn't support the latest version, choose the latest version available for your device at least. Here are some websites you can use to update popular browsers:

Google Chrome:

Update Google Chrome - Users can update Google Chrome to the latest version here.

Mozilla Firefox:

Update Mozilla Firefox - The official Mozilla site provides updates for Firefox.

Microsoft Edge:

Update Microsoft Edge - The official Microsoft Edge site has options to update the browser.

Apple Safari:

Update Apple Safari - This link allows Apple users to update Safari on their devices. Always remember to update your browsers regularly to get the latest features and security.
